## Learnings

### Design System Patterns
- All colors should come from `src/design/tokens.ts` (COLORS object)
- Typography: DM Sans for body, Playfair Display for headings
- ui-v2 components are in `src/components/ui-v2/`
- Mascot system uses tokens from `src/constants/mascot.ts`

### Common Gotchas
- Some screen files have duplicate COLORS definitions - always use tokens.ts
- Button gradients use CSS variables: --gradient-primary-button, --gradient-coral-button
- Shadows use CSS variables: --shadow-primary-button, --shadow-coral-button

### File Structure
- Pages: src/pages/
- UI Components: src/components/ui-v2/
- Design Tokens: src/design/tokens.ts
- CSS Variables: src/index.css
