# Öppna denna mapp i Cursor

**Det här är projektets huvudsakliga mapp.** Kör alltid `npm run dev` här.

- Din **`.env`** ska ligga i denna mapp (samma nivå som `package.json`).
- I Cursor: **File → Open Folder** och välj **denna mapp**.
- Kör sedan: `npm run dev`.

Om du öppnat en annan mapp (t.ex. en worktree) hittar appen inte `.env` och inloggning fungerar inte. Öppna då denna mapp istället.
