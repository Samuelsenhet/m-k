# Landing – Story-struktur (MÄÄK)

Landingen ska sälja **transformation**, inte bara **funktion**.

---

## Nuvarande risk

❌ **"Hitta matchningar"** / **"Hitta kärlek"** = funktionsfokus  
→ Låter som andra dejtingappar.

✅ **"Slipp marknadsplatsen"** / **"Relation som matchar dig"** = transformation  
→ Tydlig position: vi är annorlunda.

---

## Rekommenderad story-struktur

### 1. Hook (första raderna)

- **Undvik:** "Hitta kärlek", "Få fler matchningar", "Swipa smartare".
- **Använd:**  
  - "Slipp marknadsplatsen."  
  - "Relation som matchar dig – inte bara profilen."  
  - "Kärlek som börjar med att förstå varandra."  

→ Användaren ska direkt känna: *det här är inte Tinder*.

### 2. Problem / frustration (kort)

- Referera till det många känner: swipe-trötthet, ytlighet, "välj utseende först".
- Formulera mjukt: ingen hat på andra appar, bara tydlig kontrast.
- Exempel: "Glöm ytliga swipes" (ni har redan liknande) → kan förstärkas med en rad om *varför* det inte fungerar för många.

### 3. Transformation (vad användaren får)

- **Inte:** en lista med features (AI, video, 30 frågor).
- **Jo:** en förändring i upplevelse:
  - "Matchningar som känns meningsfulla."
  - "Samtal före bedömning."
  - "Personlighet först – så du slipper prestera på rätt saker."

→ Features kan komma *efter* transformation, som stöd ("Så gör vi det").

### 4. CTA (call to action)

- **Undvik:** "Börja swipa", "Hitta matchningar idag", "Se vilka som gillar dig."
- **Använd:**  
  - "Kom igång" / "Kom igång gratis"  
  - "Lär känna dig – och någon som passar"  
  - "Ta personlighetstestet" (om det är första steget)  

→ CTA ska kännas som ett steg in i en *resa*, inte in i en feed.

### 5. Social proof / trygghet (valfritt)

- Om ni visar antal användare: formulera som "Andra som valt att göra det annorlunda" snarare än "X användare online nu" (som kan kännas som marknadsplats).
- Mascot och ton ska förstärka trygghet och lugn.

---

## Konkret justering av nuvarande landing (förslag)

| Nu | Förslag (transformation) |
|----|---------------------------|
| "Hitta kärlek som matchar din själ" | Behåll eller byt till: "Slipp marknadsplatsen. Kärlek som matchar din själ." |
| "Glöm ytliga swipes. MÄÄK matchar dig baserat på personlighet, inte utseende." | Behåll – det är redan transformation. Kan förstärkas: "… så du kan vara dig – inte bara din profil." |
| Features (30 frågor, AI, video) | Flytta ned eller formulera som *hur* vi når transformation: "Vi använder personlighet, samtal och tid – inte swipe och siffror." |
| CTA "Kom igång gratis" | OK. Alternativ: "Kom igång – lugn och i din takt." |

---

## Ton överallt på landingen

- **Lugn**, inte stress eller FOMO.
- **Mening**, inte antal (matchningar, likes, procent).
- **Relation**, inte marknadsplats (köpare/säljare, utbud/efterfrågan).
- **Transformation**, inte feature-lista.
